# 实现操作栏下拉选项和增加任务功能

## 功能需求
1. **操作栏改为下拉选项**：将当前的单个"编辑"按钮改为下拉菜单
2. **下拉选项包含两个功能**：编辑（已实现）和增加任务（需实现）
3. **增加任务功能**：点击后弹出表单，填写任务信息并提交

## 实现步骤

### 1. 前端修改
1. **更新导入**：在 `frontend/src/components/project/ProjectPanel.tsx` 中导入 `Dropdown`、`Menu`、`Modal` 和 `Form` 组件
2. **添加状态管理**：
   - 添加 `taskModalVisible` 状态控制增加任务模态框的显示/隐藏
   - 添加 `currentProjectId` 状态存储当前操作的项目ID
   - 添加 `taskForm` 状态存储任务表单数据
3. **修改操作栏**：
   - 将操作栏的单个按钮改为下拉菜单
   - 添加"编辑"和"增加任务"两个选项
4. **实现增加任务功能**：
   - 添加增加任务模态框
   - 实现任务表单
   - 添加创建任务的 mutation
   - 实现表单提交和数据验证
5. **添加错误处理**：添加创建任务的错误处理和成功/失败消息提示

### 2. 技术实现
- **下拉菜单**：使用 Ant Design 的 `Dropdown` 和 `Menu` 组件
- **模态框和表单**：使用 Ant Design 的 `Modal` 和 `Form` 组件
- **数据提交**：使用现有的 `createTask` API 接口
- **状态管理**：使用 React 的 `useState` 管理模态框状态和表单数据
- **错误处理**：添加错误处理和成功/失败消息提示

### 3. 预期效果
- 操作栏显示为下拉菜单按钮
- 点击下拉菜单可选择"编辑"或"增加任务"
- 点击"编辑"进入编辑模式（已实现）
- 点击"增加任务"弹出任务表单
- 填写任务信息后点击提交，成功后关闭模态框并刷新项目列表

- 验证"增加任务"功能是否正常工作
